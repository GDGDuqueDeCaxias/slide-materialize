Materialize

O que é?

É um framework de CSS para Front-end. Ele é baseado no Material Design. O Material Design é um estilo de design criado pela Google, objetivando unificar a experiência do usuário em qualquer plataforma.

Mas o que é esse bendito
Material Design?!

Em que este framework pode nos ajudar?

Todo framework tem a função de agrupar funcionalidades que são comuns em várias aplicações. O Materialize não é diferente, ele vai te auxiliar no momento de criação da sua página web. O Materialize conta com uma grande gama de componentes com design baseados no Material Design prontos para uso, conta ainda com um sistema grid para criação de páginas responsivas.

Componentes presentes no Materialize

  • Botões flat
  • Cards
  • Menus resposivos
  • Parallax
  • tabs

Benefícios

  • Fácil customização dos componentes
  • Incentiva sua criatividade
  • Focado na experiência do usuário
  • Rápido desenvolvimento
